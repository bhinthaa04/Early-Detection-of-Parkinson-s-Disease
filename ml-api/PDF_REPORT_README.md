# Parkinson's Disease Screening Report - PDF Generation

This document describes the PDF report generation features added to the Flask API.

## Overview

The system now includes enhanced PDF report generation using ReportLab with:
- Professional medical report layout
- Logo at top center
- Watermark in background (low opacity)
- Comprehensive sections matching the HTML template

## New Endpoint

### POST /generate-report

Generates a PDF report from form data submitted as JSON.

**Request Body (JSON):**

```
json
{
  "patient_name": "John Doe",
  "age": 65,
  "gender": "Male",
  "patient_id": "PD12345678",
  "medical_history": "Diabetes, Hypertension",
  "current_symptoms": "Tremors in hands, slow movement",
  "family_history": "No family history of Parkinson's",
  "current_medications": "Levodopa",
  "test_type": "AI-based Multimodal Screening",
  "input_data": "Spiral Drawing Image, Voice Sample",
  "test_date": "2024-01-15 10:30:00",
  "prediction": true,
  "confidence": 0.85,
  "stage": "Early Stage (1-2)"
}
```

**Response:** PDF file download

### GET /download-report

Legacy endpoint that accepts query parameters.

**Query Parameters:**
- `prediction`: "Yes" or "No"
- `confidence`: 0-100
- `stage`: Disease stage
- `patient_name`: Patient name
- `age`: Patient age
- `gender`: Patient gender
- `patient_id`: Patient ID

**Example:**
```
/download-report?prediction=Yes&confidence=85&stage=Early+Stage&patient_name=John+Doe&age=65&gender=Male&patient_id=PD12345678
```

## PDF Sections

The generated PDF includes:

1. **Header**
   - Logo at top center
   - NeuroScan AI branding
   - Report title

2. **Patient Information**
   - Patient Name, Age, Gender, Patient ID
   - Previous Medical History
   - Current Symptoms
   - Family History
   - Current Medications

3. **Test Information**
   - Test Type
   - Input Data (Spiral Drawing Image, Voice Sample)
   - Test Date & Time

4. **Prediction Result**
   - Status (Positive / Negative)
   - Confidence Score
   - Disease Stage
   - Risk Level (Low / Moderate / High)

5. **Interpretation**
   - Dynamic text based on Positive or Negative result

6. **Recommendations**
   - Schedule neurological consultation
   - Repeat test after 3-6 months
   - Monitor symptoms

7. **Lifestyle & Health Advice**
   - Exercise regularly
   - Balanced diet
   - Sleep 7-9 hours
   - Stress management

8. **Important Notice**
   - AI-based screening disclaimer
   - Not a final medical diagnosis

9. **Footer**
   - Page numbers
   - Generated by NeuroScan AI - Parkinson's Predictor System

## Watermark

A watermark logo is drawn in the background of each page with low opacity (8%) to maintain readability while providing branding.

## Running the Server

### Option 1: Using batch file (Windows)
Double-click `start_server.bat` to start the Flask server.

### Option 2: Using command line
```
cd ml-api
pip install -r requirements.txt
python app.py
```

The server will run on http://127.0.0.1:8000

## Testing with HTML Form

Open `example_form.html` in a web browser to test the PDF generation:

1. Start the Flask server
2. Open `example_form.html` in browser
3. Fill in the form fields
4. Click "Generate PDF Report"
