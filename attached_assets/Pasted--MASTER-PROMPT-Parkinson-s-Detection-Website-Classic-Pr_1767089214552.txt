🌐 MASTER PROMPT: Parkinson’s Detection Website (Classic & Professional)

Prompt:

Design and develop a classic, professional, and trustworthy Parkinson’s Disease Detection & Awareness website, inspired by established medical organizations (such as Parkinson’s Disease & Movement Disorder societies).

🎨 Visual Style & Branding

Use a clean, medical-grade design with a calm and dignified tone.

Primary colors: soft blue, navy blue, white, light beige, and subtle teal accents.

Avoid flashy or neon colors; keep the look professional, compassionate, and inclusive.

Typography should be simple, readable, and formal (e.g., sans-serif fonts like Inter, Open Sans, or Lato).

🖼️ Hero Section (Like the Reference Image)

Full-width hero banner with a high-quality image of people united together (patients, caregivers, or community members).

Image treatment should feel hopeful and empowering, not clinical or gloomy.

Overlay a soft dark gradient for text readability.

Centered inspirational message, for example:

“Together We Move Better”

Emphasize unity, dignity, and support.

🧠 Background Design

Use subtle backgrounds throughout the site:

Light gradients (white → soft blue / beige)

Abstract neural patterns or wave lines at very low opacity

Gentle section separators instead of hard borders

Backgrounds must never distract from content.

✨ Animations & Interactions

Add soft, minimal animations:

Fade-in on page load

Slide-up for content cards and sections

Smooth hover effects on buttons (slight scale + shadow)

Use slow, calming motion—no fast or flashy transitions.

Page transitions should feel smooth and professional.

🧪 Parkinson’s Detection Section

Clear, form-based interface for prediction input.

Use card-style layout with clean spacing.

Show prediction results using:

Soft color indicators (green / amber / soft red)

Gentle reveal animation (fade + scale)

Include confidence percentage and stage explanation in plain language.

♿ Accessibility & Usability

High contrast text for readability.

Large buttons and input fields.

Respect reduced-motion preferences.

Ensure design is comfortable for elderly users and patients.

📱 Responsiveness

Fully responsive design for mobile, tablet, and desktop.

Mobile layout should preserve readability and calm visual flow.

🧭 Navigation & Layout

Simple top navigation with logo on the left.

Sections such as:

Home

About Parkinson’s

Detection Tool

Awareness / Resources

Contact

Use clear spacing and structured layout.

🏥 Overall Tone

The website should communicate:

Trust

Medical credibility

Hope and community support

Suitable for patients, families, healthcare professionals, and academic evaluation.

⚙️ Tech Stack (Optional)

Modern frontend (HTML, CSS, React, or Next.js)

Tailwind CSS or clean custom CSS

Framer Motion or CSS animations (minimal use)

Optimized for performance and accessibility
